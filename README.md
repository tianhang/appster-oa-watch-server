# appster-oa-watch-server

#### Check for Node and npm
Make sure that you've installed Node and npm before attempting to install gulp.

```sh
node --version
```
```sh
npm --version
```

#### Install the `gulp` command

```sh
npm install --global gulp-cli
```

#### Clone & install

```sh
git clone xxx
```
```sh
 npm install
```

#### Run
```sh
gulp
```
